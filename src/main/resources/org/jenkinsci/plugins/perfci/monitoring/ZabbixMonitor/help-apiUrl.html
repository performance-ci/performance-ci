Usually "http(s)://zabbix-server/zabbix/api_jsonrpc.php". Both HTTP and HTTPS are acceptable.
For HTTPS connections, make sure your CA is trusted by your Jenkins server, or the outgoing connections will be refused.
If you are using self-signed certificates, import your CAs to the keystore of your Java installation. See http://docs.oracle.com/javase/8/docs/technotes/tools/unix/keytool.html.
